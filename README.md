# 🌌 SINGULARITY-X — Sci-Fi Vortex Engine 🌀

Интерактивный и высокооптимизированный математический визуализатор, имитирующий физику поведения космической сингулярности, гравитационных вихрей и кастомных геометрических аномалий.

## 🌐 Живое демо (Live Demo)
👉 **[Запустить проект в браузере](https://21nizom.github.io/SINGULARITY-X)**

## 🚀 Ключевые фишки (Features)
- **Динамическая мутация форм:** Математическая модель плавно трансформируется между различными состояниями — от сверхплотного кольца частиц (зелёный тор) до сложной многоугольной фрактальной сетки (розовый вихрь).
- **Киберпанк HUD-интерфейс:** Интегрированная система мониторинга со Sci-Fi индикаторами состояний ядра, включая телеметрию: `VORTEX V.5`, `STATUS: MAXIMUM`, `HYPER_OPTIMIZED`.
- **Генеративная графика:** Рендеринг сотен процедурно связанных точек в реальном времени с сохранением высокой частоты кадров (FPS).
- **Интерактивное управление:** Возможность динамически менять состояния и цвета аномалии при взаимодействии с экраном.

## 🛠️ Технологический стек (Tech Stack)
- HTML5 / CSS3 (Кастомный футуристичный интерфейс бортового компьютера)
- JavaScript (ES6+ / Математические алгоритмы смещения вершин и частиц)
- **Three.js / WebGL / Canvas API** (Процедурная генерация 3D-геометрии и рендеринг эффектов свечения)
